## 版本号格式

版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 规范：

- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正
